How to Install and Run

1

Download ZIP

Click "Download Extension ZIP" to download the packaged extension archive directly onto your machine.

2

Extract the ZIP

Locate the downloaded zip file and extract it into a folder (e.g. fb-warmup-extension).

3

Load Unpacked

Open Chrome, navigate to chrome://extensions, enable "Developer mode" (top right), and click "Load unpacked".

4

Start Warm Up
