How to Install WarmUp Bot

Follow this step-by-step developer guide to quickly load, run, and activate the Facebook Account Warm Up Chrome Extension in less than 3 minutes.

1

Download the Extension Bundle

Click on the download button anywhere on this website to save the packaged extension zip archive (fb-warmup-extension.zip) directly onto your desktop or computer.

This ZIP file contains the full source code (manifest, content script, interactive popup screen, announcements fallbacks, and the branding logo asset) required for the extension to operate locally.

2

Extract the Zip Archive

Once downloaded, locate the archive file and extract its content to a specific folder on your local drive. You can name the destination folder fb-warmup-extension or something easy to find.

  • On Windows: Right-click the file and select Extract All..., then follow instructions.
  • On Mac: Double-click the ZIP archive to instantly decompress it.
3

Enable Developer Mode in Chrome

Chrome Extensions loaded locally must be processed under Chrome developer rules. Open your Chrome browser, click the three-dot menu at the top right, navigate to Extensions, and choose Manage Extensions. Alternatively, simply paste this link in your browser bar: chrome://extensions.

Locate the Developer mode toggle button at the top-right corner of the Extensions dashboard page and slide it to Enabled (ON).

4

Load the Unpacked Directory

Once developer mode is enabled, click the "Load unpacked" button located at the top left of the chrome extensions manager dashboard page.

Browse your filesystem, select the directory/folder where you extracted the ZIP archive in Step 2, and click Select Folder. The WarmUp Bot extension will now appear in your active extensions list.

5

Pin and Run the Extension

Click on the extensions puzzle piece icon in your Chrome toolbar, locate WarmUp Bot, and click the pin icon to make it always visible.

Now open a new browser tab, go to facebook.com, log into your profile, click the extension icon to reveal the control panel, configure your warm-up preferences, and click Start Warm Up Session!
