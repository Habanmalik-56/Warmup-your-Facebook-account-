Safety-First Features

WarmUp Bot uses advanced simulation algorithms to emulate true human browser interaction, keeping your Facebook accounts completely safe and trustworthy.

🖱️
Human Behavior

Smooth & Variable Scrolling

Unlike standard web automation scripts that scroll in fixed, artificial patterns, WarmUp Bot scrolls page-by-page using dynamic, random pixel steps. It accelerates and decelerates naturally, mimicking how a human would physically spin a mouse wheel or swipe on a trackpad.

This completely random pattern prevents Facebook algorithms from flagging the account behavior as automated or programmatic.

👍
Engagement Simulation

Sequential Post Interactions (Auto Liking)

WarmUp Bot integrates a robust feed post identifier. As it scrolls, it tracks which posts have already been interacted with and targets unliked, visible posts in the browser window.

By liking posts sequentially with randomized stabilization delays (between 150ms to 300ms), the system ensures a high level of active profile trust without spamming clicks too rapidly.

⏱️
Safety Algorithm

Realistic Reading Pauses

Real users stop to read posts, watch video clips, or examine images. WarmUp Bot mimics this behavior by calculating random scrolling distance thresholds (between 3600px and 7200px).

Once reached, it temporarily halts all scrolling for a random period (1 to 2.5 seconds) to simulate a user's attention span, before smoothly resuming feed scrolling.

⚡
Speed Customization

Speed Profile Selectors

Choose between three pre-configured speeds based on your Facebook account's current age, trust factor, and warm-up requirements:

Slow: Best for brand-new accounts or fresh profiles with zero previous activity.Normal: The standard human-like activity configuration.Fast: Fast scrolling speed designed to speed up the warm-up cycle on matured accounts.

📊
Account Statistics

Live Tracking Dashboard

The dark-themed extension control popup shows live telemetry statistics. Monitor your session duration progress bar, exact count of successful scrolls triggered, and total posts liked during the current active warm-up session.

🔄
Crash Protection

Auto-Resume Session Pipeline

If you refresh the Facebook page, experience a network drop, or navigate between posts, WarmUp Bot automatically picks up the session right where it left off, utilizing local memory storage to continue running without manual restart.
